# AC Services Website (Next.js)

## Run locally
1) Copy env file:
   - Copy `.env.local.example` to `.env.local` and update numbers.

2) Install deps:
```bash
npm install
```

3) Start dev server:
```bash
npm run dev
```

Open:
- http://localhost:3000
- http://localhost:3000/contact
- http://localhost:3000/admin (Basic Auth)

Leads are stored in `data/leads.json`.
